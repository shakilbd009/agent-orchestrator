# UI Domain

This directory contains UI-specific BRDs for the Agent Orchestrator Platform.

BRDs in this directory address user-facing interface requirements including the dashboard, agent workstream views, and notification surfaces.

## Structure

| File | Domain | Phase | Status |
|------|--------|-------|--------|
| brd-03-client-portal.md | Client Portal and Business Project Board | Phase 1 | Approved for Curation |
| BRD-03-dashboard.md | Business Dashboard placeholder | Phase 1 | superseded by `brd-03-client-portal.md` |
| BRD-04-agent-dashboard.md | Agent Workstream View | Phase 1 | placeholder |

## Guidelines

- UI BRDs use plain-language descriptions without CSS, design tokens, or rem units.
- Layout and spacing requirements are expressed as prose, not implementation syntax.
- Component specifications defer to Phase 1 UI BRDs after the app shell exists.
